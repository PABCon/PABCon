## Instalations

### Brew Instalation (MAC)

```/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"```
[Brew Website](https://brew.sh/)

### Node

```$ brew install node```

### Firebase

```npm install firebase```
